작업공간 이동하는 방법
1. 특정폴더에 내 워크스페이스를 만든다.
   (ex. d:\\csm\workspace)

2. 기존작업한 프로젝트를 복사하여 1번폴더에 붙여놓기

3. 이클립스에서 file -> switch workspace -> other
   1번 폴더를 선택

4. 이클립스에서 file -> import
   general -> exiting project into workspace

<get방식>
데이터가 보이니까 연습할때는 편함, 제한X
