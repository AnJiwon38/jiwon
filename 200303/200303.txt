4.명명규칙
  1) 대소문자 구분, 길이에 제한 X
     ex.True와 true는 다름
     -> 길게 하는건 나중에 헷갈려서 좋지않음
  2) 예약어 사용 X
     ex. true는 예약어라 사용 X, True는 사용 O
     -> 굵고 색상이 다르게 나오는 문자
  3) 숫자로 시작 X
  4) 특수기호는 '_'와 '$'만 사용가능
4-1. 권장사항
  1) 클래스 이름의 첫 글자를 항상 대문자
     - 변수와 메서드 이름의 첫 글자는 항상 소문자
  2) 여러 단어 이름은 단어의 첫 글자를 대문자
  3) 상수의 이름은 대문자. 단어는 '_'로 구분

5.변수, 상수, 리터럴
  1) 변수 - 하나의 값을 저장하기 위한 공간, 또 다른 값이 다시 들어갈 수 있음. 
  2) 상수 - 한 번만 값을 저장할 수 있는 공간. 
  3) 리터럴 - 그 자체로 값을 의미하는 것. 변수 또는 상수에 들어가는 값
    int score = 100;
        score = 200;
      char ch = 'A';
   String str = "abc";
final int MAX = 100;   //MAX 상수, 대문자로 사용. MAX 값을 100으로 지정해줌.
왼쪽은 변수, 상수. 오른쪽은 리터럴
